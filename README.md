# [cite_start]EagleRC's System Cleaner [cite: 8, 9]

[cite_start]EagleRC's System Cleaner is a lightweight, automated Windows batch utility designed to optimize your PC[cite: 9, 15]. [cite_start]It safely removes useless temporary files and clears system caches to free up space and maintain optimal performance[cite: 2, 14].

## 🚀 Features

* [cite_start]**Automated Admin Check:** The script automatically detects if it has the required permissions and will gracefully restart itself with Administrator privileges if needed[cite: 1, 2].
* [cite_start]**Interactive Menu:** Features a simple command-line interface allowing you to easily start the cleaning process or exit[cite: 10, 11].
* **Comprehensive Junk Removal:** Efficiently deletes files and subdirectories from:
  * [cite_start]User Temp folder (`%temp%`) [cite: 13]
  * [cite_start]Windows Temp folder (`C:\Windows\Temp`) [cite: 13]
  * [cite_start]System Prefetch folder (`C:\Windows\Prefetch`) [cite: 13]
  * [cite_start]Recent files history (`%AppData%\Microsoft\Windows\Recent`) [cite: 13]
* [cite_start]**Network Optimization:** Flushes the DNS cache to resolve potential connectivity issues and clear outdated network data[cite: 13].
* [cite_start]**Custom UI:** Includes a custom animated ASCII art banner upon launch for a stylized terminal experience[cite: 3, 17].

## 🛠️ How to Use

1. Download the `Cleaner.bat` file to your Windows machine.
2. Double-click the file to run it.
3. [cite_start]If prompted by User Account Control (UAC), click **Yes** to grant Administrator privileges (required for cleaning system folders like Prefetch and Windows Temp)[cite: 1, 2].
4. [cite_start]Wait for the animated banner to load[cite: 3, 17].
5. [cite_start]Press `1` to start the system clean, or `0` to exit[cite: 10, 11].
6. [cite_start]The script will automatically wipe the targeted directories and display a success message when your PC is optimized[cite: 13, 14, 15].
7. [cite_start]Press any key to close the window[cite: 17].

## ⚠️ Requirements

* [cite_start]**Permissions:** Administrator rights are required for full functionality[cite: 1].
